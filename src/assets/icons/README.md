# SVG Icons from Figma

This folder contains SVG icons exported from Figma designs.

## Icons Required

Based on the Figma designs, the following icons are needed:

1. **chevron-down** - Dropdown indicator ✓ (implemented in Icon component)
2. **map-pin** - Location selector ✓ (implemented in Icon component)
3. **briefcase** - Job title field ✓ (implemented in Icon component)
4. **arrow-right** - Next/Continue button ✓ (implemented in Icon component)
5. **arrow-left** - Back button ✓ (implemented in Icon component)
6. **check** - Selected state ✓ (implemented in Icon component)
